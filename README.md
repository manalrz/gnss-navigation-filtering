# GNSS Navigation & Sensor Fusion

Two MATLAB labs on satellite and inertial navigation, covering position estimation, filtering, and multi-sensor fusion techniques.

## Projects

### [`TP GPS satellite`](./TP%20GPS%20satellite) — GPS Satellite Navigation
Position estimation from GPS pseudorange measurements via nonlinear least squares (Gauss-Newton), with robustness analysis against interference and multipath, and precision characterization via Dilution of Precision (DOP).

### [`TP IRS Hybridation`](./TP%20IRS%20Hybridation%20) — GPS/INS Sensor Fusion
2D strapdown inertial mechanization combined with GPS via a linearized Kalman filter, comparing open-loop and closed-loop hybridization and testing resilience to simulated GPS outages.
